# Unit 6 — Box Model Practice

## Task 1 — Style Your Code to Match the Board

Use the following CSS properties to complete the styling:

- text-align
- border
- border-style
- border-color
- color
- font-size
- background-color
- padding
- border-top-style
- border-bottom-style
- margin
- width
- height
- border-radius

You will need to write **5 CSS rules** in total. These are the tags you will need to target:

- header
- p
- footer
- body
- img

The challenge is figuring out:

1. Which properties belong inside each rule
2. How many properties go into each rule
    - Hint: the `img` tag uses 9 properties — the others use less.

Good luck! 🎯
